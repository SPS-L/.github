## About Us

The **Sustainable Power Systems Laboratory (SPS-Lab)** is led by [Dr. Petros Aristidou](https://sps-lab.org) and is part of the **Department of Electrical Engineering and Computer Science and Engineering** at the [Cyprus University of Technology (CUT)](https://cut.ac.cy), Limassol, Cyprus.

We work on making future electric power systems **sustainable, secure, and resilient**. Our research bridges mathematical tools from numerical analysis and optimization with high-performance computing and machine learning to tackle modern power system challenges.

---

## 🔬 Research Areas

- **Power System Dynamics & Stability** — Transient stability, small-signal analysis, low-inertia grids
- **Real-Time & Time-Domain Simulation** — Advanced simulation tools for large-scale power systems
- **Renewable Energy Integration** — RES hosting capacity, curtailment strategies, frequency security
- **Smart Grids & Active Distribution Networks** — Volt-Var optimization, EV flexibility, demand response
- **Machine Learning for Power Systems** — Data-driven control, safe multi-agent reinforcement learning
- **Energy Planning & Optimization** — District heating/cooling, virtual power plants, EMS

## 🛠️ Key Open-Source Tools & Projects

| Repository | Description | Language |
|---|---|---|
| [stepss-PyRAMSES](https://github.com/SPS-L/stepss-PyRAMSES) | Time-domain dynamic simulator for electric power systems | Python |
| [STEPSS](https://github.com/SPS-L/STEPSS) | Static and Transient Electric Power Systems Simulation platform | Java |
| [stepss-URAMSES](https://github.com/SPS-L/stepss-URAMSES) | RAMSES core simulation engine | Fortran |
| [RAMSES-Eigenanalysis](https://github.com/SPS-L/RAMSES-Eigenanalysis) | Eigenanalysis toolbox for RAMSES | MATLAB/Octave |
| [HeatPump-FlexModeling-Toolkit](https://github.com/SPS-L/HeatPump-FlexModeling-Toolkit) | Flexibility modeling for heat pumps in energy systems | Jupyter |
| [Cyprus-Grids](https://github.com/SPS-L/Cyprus-Grids) | Open data and models for the Cyprus power grid | Jupyter |
| [TSOC-data-analysis](https://github.com/SPS-L/TSOC-data-analysis) | Analysis tools for Cyprus TSOC operational data | Python |

> 💡 Several repositories are private (active R&D). Public repos are listed above.
